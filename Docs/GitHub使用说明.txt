GitHub从本地上传到GitHub服务器
	1.在gitHub上新建项目名XXX
	2.下载项目 git clone https://github.com/QionghuaH/XXX.git
	3.在本地 将需要添加的文件加入XXX目录下
	4.本地执行 git add .   (. 可以是某个文件)
	5.本地执行  git commit -m "first commit"  显示备注first commit
	可选: git config --global user.email "you@example.com"
        git config --global user.name "Your Name
	6.git push -u origin master
  
  
  git pull
